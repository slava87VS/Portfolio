# Задачи проекта #

На основе статистики о платёжеспособности клиентов исследовать влияет ли семейное положение и количество детей клиента на факт возврата кредита в срок

# Инструменты #

- предобработка данных
- Python
- Pandas

# Выводы #

На основе данных кредитного отдела банка исследовал влияние семейного положения и количества детей на факт погашения кредита в срок. Была получена информация о данных. Определены и обработаны пропуски. Заменены типы данных на соответствующие хранящимся данным. Удалены дубликаты. Категоризованы данные. Один датафрейм декомпозирован на три. Проведена глубокая аналитика и сделаны выводы касаемо характеристик платежеспобных заемщиков*
